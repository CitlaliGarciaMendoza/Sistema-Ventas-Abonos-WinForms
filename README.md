# Sistema de Control de Ventas y Abonos

## Descripción

Aplicación de escritorio desarrollada en C# y .NET WinForms para la gestión de clientes, productos y control de ventas con pagos en parcialidades (abonos).

Este proyecto simula un sistema real utilizado en pequeños negocios para llevar el control de ventas y seguimiento de pagos pendientes.

## Funcionalidades

* Registro de clientes
* Registro de productos
* Control de ventas
* Registro de abonos (pagos parciales)
* Cálculo automático de saldo pendiente
* Generación de reportes en PDF

## Tecnologías

* C#
* .NET Framework (WinForms)
* SQL Server
* Stored Procedures

## Estructura del sistema

* Clientes
* Productos
* Ventas
* Abonos
